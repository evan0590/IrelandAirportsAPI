Ireland Airport API's information page

Users can retrieve departures and arrivals data from a selection of Irish airports:

  • Dublin (DUB)
  • Shannon (SNN)
  • Cork (ORK)

The following functions are available

1. Retrieve All Information - usage:
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com/arrivals returns JSON array of all information from the arrivals table with airport, airport_iata, airline_name, flight_iata, flight_date, arrival_scheduled, departure_airport, departure_scheduled.
OR
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com/departures returns JSON array of all information from the departures table with airport, airport_iata, airline_name, flight_iata, flight_date, departure_scheduled, arrival_airport, arrival_scheduled.

2. Retrieve Information by Flight IATA - example:
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com/arrivals/iata/EY45 returns JSON object from the arrivals table with airport, airport_iata, airline_name, flight_iata, flight_date, arrival_scheduled, departure_airport, departure_scheduled filtered by flight_iata (Internation Air Transport Association) code. The same approach applies for the departures route.

3. Retrieve Information on a given Date - example:
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com:80/arrivals/date/2021-02-20 returns JSON array from the arrivals table with airport, airport_iata, airline_name, flight_iata, flight_date, arrival_scheduled, departure_airport, departure_scheduled filtered by flight_date. The same approach applies for the departures route.

4. Retrieve Information by Airport - example:
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com:80/arrivals/airport/SNN returns JSON array from the arrivals table with airport, airport_iata, airline_name, flight_iata, flight_date, arrival_scheduled, departure_airport, departure_scheduled filtered by airport_iata. The same approach applies for the departures route.

5. Retrieve Information by Airline - example:
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com:80/arrivals/airline/Aer%20Lingus returns JSON array from the arrivals table with airport, airport_iata, airline_name, flight_iata, flight_date, arrival_scheduled, departure_airport, departure_scheduled filtered by airline_name. The same approach applies for the departures route.

6. Retrieve Information by Airport on a given Date - example:
http://ec2-52-19-19-167.eu-west-1.compute.amazonaws.com:80/arrivals/airportdate/DUB/2021-02-20 returns JSON array from the arrivals table with airport, airport_iata, airline_name, flight_iata, flight_date, arrival_scheduled, departure_airport, departure_scheduled filtered by airport_iata and flight_date. The same approach applies for the departures route.

Please note all these webservice names and parameters are case sensitive
